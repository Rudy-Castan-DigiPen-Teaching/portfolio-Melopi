Project Procedural Geometric Modeling

3D Graphics

Project Overview

The objective of this assignment is to implement procedural mesh generation algorithms using C++ and computer graphics mathematics, such as trigonometry and matrix transformations. Starting from basic primitives like a Plane and a Cube, the project extends to dynamically computing the vertices, normal vectors, and UV texture coordinates for complex 3D geometries, including a Sphere, Torus, Cylinder, and Cone. This procedural approach allows for real-time shape rendering and structural modification without relying on pre-baked external asset files.

Tasks Completed

  • Index Buffer & Plane Foundation: Implemented the build_index_buffer and create_plane_vertices functions to establish a solid foundation for reusing vertex data efficiently and optimizing memory allocation.
  • Procedural Mesh Generation:
    • Cube: Generated six individual plane faces and mathematically assembled them into a closed 3D cube using precise translation and rotation transformation matrices.
    • Sphere & Torus: Utilized spherical coordinate systems and trigonometric functions to dynamically calculate smooth surface geometry, accurate outward-facing normals, and proper UV mapping.
    • Cylinder & Cone: Developed the procedural body generation along with a dedicated capping algorithm (add_cap) to seal the top and bottom openings with correct texture coordinate scaling.
  • Wireframe Topology Conversion: Created the convert_to_lines_pattern utility function to transform index arrays from triangle primitives into distinct line segments, enabling clean wireframe visualization and structural debugging.

Reflection

The primary challenge of this project lay not only in formulating the mathematical equations for complex 3D primitives but also in satisfying the exceptionally strict type safety requirements enforced by the project's compiler configuration. Fine-tuning implicit type promotions between float/double and resolving signed/unsigned mismatches between int/size_t provided a profound debugging experience. Ultimately, witnessing abstract mathematical representations successfully transition into interactively rendered 3D assets inside a web browser deeply reinforced my understanding of the procedural modeling pipeline.
