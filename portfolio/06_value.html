Project Value Noise

3D Graphics

Project Overview

The primary objective of this project was to implement a robust Value Noise algorithm in C++ for generating procedural textures. This involved integrating the noise generation logic into a custom graphics framework and utilizing OpenGL shaders to correctly visualize the resulting textures across multiple dimensions.

Tasks Completed

  • Implemented 1D, 2D, and 3D value noise algorithms in C++ utilizing glm::vec4 data types.
  • Developed a permutation hashing system with Fisher-Yates shuffling for efficient and randomized noise computation.
  • Applied adjustable interpolation methods (Linear, Bilinear, and Trilinear) combined with customizable smoothing functions (e.g., Quintic).
  • Extended the texture management system (Texture::UploadAsRGBA) to dynamically upload generated pixel data to the GPU.
  • Created specific vertex and fragment shaders to scale, interpolate, and map procedural textures seamlessly onto geometry.

Reflection

This project solidified the mathematical concepts behind procedural generation, particularly the accurate calculation of noise coordinates and the application of multi-dimensional interpolation. A key technical challenge was ensuring the precise alignment of data structures between the CPU noise generation and the GPU texture rendering pipeline. Debugging texture scaling and coordinate mapping in the shaders highlighted the critical importance of verifying data flow across the CPU-GPU boundary. The completed implementation successfully generates seamless, adjustable noise patterns while maintaining stable cross-platform performance.
